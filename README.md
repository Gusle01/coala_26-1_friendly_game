# coala_26-1_friendly_game

`A안(점수 레이스형)` MVP 웹 버전입니다.

## 실행 방법
1. `index.html`을 브라우저에서 엽니다.
2. 팀을 등록합니다.
3. 활동 기록을 입력해 점수를 누적합니다.

## 점수 규칙
- 기본 점수: 활동 1회당 `10점`
- 난이도 보너스:
  - 쉬움 `+10점`
  - 보통 `+20점`
  - 어려움 `+30점`
- 참여 팀원 수 보너스:
  - `1~2명`: 추가 점수 없음
  - `3명 이상`: 1명 추가마다 `+20점`
- 세트 보너스: 카테고리 4개를 모두 1회 이상 완료하면 `+40점` (1회)
- 라인 보너스: 한 카테고리 3회 달성마다 `+25점`

## 포함 기능
- 팀 등록
- 활동 기록 및 점수 자동 계산
- 실시간 랭킹
- 레이스 트랙 UI(깃발까지 진행도 표시)
- 최근 활동 로그
- 로컬 저장(LocalStorage) 및 초기화
