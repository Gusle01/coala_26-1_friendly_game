# board_A_python

친해지길 바라 `A안(점수 레이스형)` Python GUI 버전입니다.

## 실행 방법
```bash
cd board_A/board_A_python
python3 gui.py
```

## 규칙
- 기본 점수: +10
- 난이도 보너스: 쉬움 +10 / 보통 +20 / 어려움 +30
- 참여 인원 보너스: 3명 이상부터 1명당 +20
- 세트 보너스: 4개 카테고리(대화/협동/도움/도전) 최소 1회씩 달성 시 +40 (1회)
- 라인 보너스: 같은 카테고리 3회마다 +25

## 포함 기능
- 팀 등록
- 활동 기록 및 자동 점수 계산
- 실시간 랭킹
- 점수 레이스 시각화(깃발)
- 최근 로그
- 로컬 저장(state.json) 및 초기화
