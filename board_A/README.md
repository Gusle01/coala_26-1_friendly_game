# board_A

친해지길 바라 `A안(점수 레이스형)` 웹 버전입니다.

## 실행
브라우저에서 `board_A/index.html`을 열면 됩니다.

## 핵심 규칙
- 기본점수 10점 + 난이도 보너스(쉬움 10 / 보통 20 / 어려움 30)
- 참여 팀원 수 보너스: 3명 이상부터 1명당 +20점
- 카테고리 4종 완료 세트 보너스 +40점(1회)
- 카테고리 3회마다 라인 보너스 +25점

## 기능
- 팀 등록
- 활동 기록 및 자동 점수 계산
- 실시간 랭킹
- 깃발 레이스 UI
- 활동 로그 / 로컬 저장 / 초기화
