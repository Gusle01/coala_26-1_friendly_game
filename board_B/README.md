# board_B

친해지길 바라 `B안(윷 레이스형)` 웹 버전입니다.

## 실행
브라우저에서 `board_B/index.html`을 열면 됩니다.

## 게임 흐름
1. 팀 등록
2. 활동 기록으로 점수 획득
3. 윷 던지기로 트랙(50칸) 이동
4. 상대 팀 잡기/완주 보너스로 역전

## 점수/보너스
- 활동 점수: 기본 10 + 난이도(10/20/30)
- 참여 인원 보너스: 3명 이상부터 1명당 +20
- 던지기 지급: 누적 점수 40점마다 던지기 기회 +1
- 세트 보너스: 카테고리 4종 1회 이상 달성 시 +40 (1회)
- 라인 보너스: 카테고리 3회마다 +25
- 잡기 보너스: 상대 말 잡으면 팀당 +30 + 추가 던지기 +1
- 잡힌 팀 복귀: 시작점이 아닌 마지막 10칸 체크포인트로 복귀
- 완주 보너스: +50

## 윷 결과
- 도: +1칸
- 개: +2칸
- 걸: +3칸
- 윷: +4칸, 추가 던지기 +1
- 모: +5칸, 추가 던지기 +1
- 빽도: -1칸

## 포함 기능
- 팀 등록
- 활동 기록 / 자동 점수 계산
- 윷 던지기 / 트랙 이동
- 상대 말 잡기
- 실시간 랭킹
- 로그 / 로컬 저장 / 초기화
